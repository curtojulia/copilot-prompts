## 💻 Sobre o Projeto

Vamos utilizar um modelo de LLM para criar dois agentes, agente 1 - te auxilia para descobrir qual carreira mais combina com você e o agente 2- cria um plano de estudo baseado em todas suas respostas

Nos códigos de AGENT 1 - Entrevistador de carreira e AGENT 2 - Planejador de Carreiras, são passados os prompts que deverão ser passados para um modelo de inteligência artificial.
Nos códigos AGENT 1 - resposta e AGENT 2 - resposta, serão passadas as respostas dos LLMs, da Inner AI.

## 📚 Pré-requisitos de Habilidades e Níveis de Conhecimento

Antes de ingressar neste conteúdo, é necessário possuir conhecimento prévio nas seguintes áreas:

- [engenharia de prompts] | [Intermediário]

## 🎯 Objetivos e Resultados Esperados

Após a conclusão do curso/projeto, os estudantes estarão aptos a:

- [Entender como modelos de linguagem pensam, evitar alucinação em razão do prompt passado]
